How to run Rinde application?

Prepare Python script

First of all, create __init__.py file and write following code:import rinde

if __name__ == '__main__':
    rinde.Application("Window title", "scene_directory")Subsequently, create new directory next to script and name it as scene_directory. You will keep there few files that defines Stage's layout and appearance. Notice that directory name has to be the same as in script.

Prepare scene directory

Scene directory has to contain two files:

layout.xml

Example code defines Scene with TextButton as its only component:<?xml version="1.0" encoding="UTF-8" ?>

<?import rinde.stage.node.button.TextButton ?>

<Scene controller="controller.Controller" width="320" height="480">
    <TextButton text="Click to print Hello" action="say_hello" id="blabla" style_class="abc" position_x="10" />
</Scene>First line informs, that file is based on XML standard. Next line forces the scene builder to import TextButton module. You don't have to import Scene module because it is imported by default. Every Stage must be controlled by its own script named as Controller so pass the path to Controller's module as argument to the Scene tag and define Window size in pixels. After that, describe content of the Stage. Look into TextButton Documentation site to see arguments that can be passed in definition.
You can access every (identified by id attribute) node from Controller script. Access to component's appearance can be simplified by style_class attribute (you will be able to refer to it in CSS file).

style.css

Defines nodes style in CSS language. Created TextButton can be stylized using these rules:/* Using selector */
text-button {
    ...declarations...
}

/* Using identifier */
#blabla {
    ...declarations...
}

/* Using style_class */
.abc {
    ...declarations...
}

Every component type has its own (mentioned in component's Documentation site) selector. Furthermore, you can stylize component by its identifier and style class (defined in layout file).

Scene controller

Controller is the object the controls Scene. Create new Python script next to __init__.py, name it as controller.py and write following code:from rinde.scene import ControllerBase

# Controller must be subclass of rinde.scene.ControllerBase
class Controller(ControllerBase):

    # Method invoked when stage gets ready
    def start(self):

        # Access to node with id="blabla" (id is defined in XML)
        print self.nodes["blabla"]

    # Method invoked when button with action="say_hello" (defined in XML) is clicked
    def say_hello(self):
        print "Hello"

Start the application

You can start the main script (__init__.py) to see the application. You can look into example projects that are distributed with library.
