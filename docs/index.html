How to run Rinde application?

Prepare Python script

First of all, create new file, name it as __init__.py and write the following code:import rinde

if __name__ == '__main__':
    rinde.Application("Window title", "scene_directory")Subsequently, create new directory next to the script and name it as scene_directory. You will keep there a few files that define layout and appearance of the stage. Notice that directory name has to be the same as in the script.

Prepare scene directory

Scene directory has to contain two files:

layout.xml

This code defines Scene with TextButton as its only component:<?xml version="1.0" encoding="UTF-8" ?>

<?import rinde.stage.node.button.TextButton ?>

<Scene controller="controller.Controller" width="320" height="480">
    <TextButton text="Click to print Hello" action="say_hello" id="blabla" style_class="abc" position_x="10" />
</Scene>The first line informs that the file is based on XML standard. Next line makes scene builder to import TextButton module. You don't have to import Scene module because it is imported by default. Every stage must be controlled by its own script, so pass the path to controller module as argument and define window size in pixels. After that, describe content of the stage. Look into TextButton documentation site to see arguments that can be passed in definition.
You can access every (identified by id attribute) node from the Controller. Access to component's appearance can be simplified by style_class attribute (you will be able to refer to it in CSS file).

style.css

Defines nodes style in CSS language. Created TextButton can be stylized using these rules:/* Using selector */
text-button {
    ...declarations...
}

/* Using identifier */
#blabla {
    ...declarations...
}

/* Using style class */
.abc {
    ...declarations...
}

Every component type has its own (mentioned in suitable documentation site) selector. Furthermore, you can stylize component by its identifier and style class.

Scene controller

Controller is an object the controls scene. Click here to open suitable tutorial site. Create new Python script next to the __init__.py, name it as controller.py and write the following code:from rinde.scene import ControllerBase

# Controller must be a subclass of rinde.scene.ControllerBase
class Controller(ControllerBase):

    # Method invoked when stage gets ready
    def start(self):

        # Access to node with id="blabla" (id is defined in XML)
        print self.nodes["blabla"]

    # Method invoked when button with action="say_hello" (defined in XML) is clicked
    def say_hello(self):
        print "Hello"

Start the application

You can start the main script (__init__.py) to run the application. You can also look into example projects that are distributed with the library. Example projects are ready-to-use and shows the basics of using Rinde.
