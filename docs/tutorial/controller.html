Controller

What is the Controller?

It is a special object that handles all logic of your application. Controller responds to the user input and performs interactions. Not only has it access to stage nodes, but also allows you to change window properties.

Methods

All you need to adjust behaviour of the application is to extend predefined methods:
MethodDescription
start(self)Invoked when the stage gets ready.
update(self)Invoked 60 times per second. Enables you to update the application regularly.
key_pressed(self, code, char)Invoked when user presses a keyboard button.
key_released(self, code)Invoked when user releases a keyboard button.
close(self)Invoked when application gets closed.

Fields

A controller has also access to some useful fields:
NameDescription
windowThis object enables you to change window's title, favicon, cursor and stage.
nodesThis is a dictionary that stores references to components. The key is an identifier of the component.
groupsThis is a dictionary that stores groups. The group is a collection of components classified by the same group attribute; handles references to selected RadioBoxes' names. The key is a group name.
