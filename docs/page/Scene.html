Scene

Scene is a proxy layer between window and content. Every node that will be shown in window has to be mentioned in a DOM structure.
Window size is determined by scene's size, so you have to pass dimensions as arguments.

Import

This module is imported by default, so you don't have to import it on your own.

XML

Obligatory attributes:
width
height
Optional attributes:
controller

Example

<Scene controller="package.ControllerClass" width="360" height="480">
    <Label text="Scene" />
</Scene>
