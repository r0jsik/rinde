Fullscreen

Fullscreen is a proxy layer between window and content. Every node that will be shown in window has to be mentioned in a DOM structure.
Window size is determined by screen's size, so you can't pass dimensions as arguments.

Import

This module is imported by default, so you don't have to import it on your own.

XML

Optional attributes:
controller

Example

<Fullscreen controller="package.ControllerClass">
    <Label text="Fullscreen" />
</Fullscreen>
