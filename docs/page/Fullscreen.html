Fullscreen

Fullscreen is a proxy layer between a window and components. Moreover, it is intended to be a root element of some XML documents, so every node that will be shown in the window has to be its child. In case of using Fullscreen, window size is determined by screen's dimensions. Fullscreen is similar to Scene and acts as a view layer in the MVC pattern. You have to link it with a controller.

Import

This module is imported by default, so you don't have to import it on your own.

XML

Optional attributes:
controller

Example

<Fullscreen controller="package.ControllerClass">
    <Label text="This window covers the entire screen area" />
</Fullscreen>
