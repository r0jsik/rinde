TextField

TextField enables the user to write some text inside. If you want to replace typed text by dots, use PasswordField instead. Its content is stored in the text property. You can set the placeholder, which will be shown when nothing typed in.

Import

Module has to be imported as <?import rinde.stage.node.field.TextField ?>.
