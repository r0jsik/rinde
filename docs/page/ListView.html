ListView

ListView is a widget which allows the user to select one of several options. The options are arranged one after another and it's impossible to select multiple options together. The component is shown when ChoiceBox's disposer clicked. Notice, that Option is the only node type the ListView can contain (it is possible to extend this class). To get selected option, use the Controller's group field.

Import

Module has to be imported as <?import rinde.stage.node.list.ListView ?>.

Obligatory attributes:
group

Example

<ListView group="language">
    <Option text="English" name="en" />
    <Option text="Polish" name="pl" selected="true" />
    <Option text="Russian" name="ru" />
    <Option text="Spanish" name="es" />
</ListView>
