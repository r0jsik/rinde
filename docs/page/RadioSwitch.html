RadioSwitch

RadioBox allows the user to switch between many options, but both of them can't be selected together. Options are grouped by the common group attribute. Every node in the group has to be named differently. You can get selected option's name using groups field in a controller. The component can display its own label, which can be moved by the spacing property and aligned left or right by the align property.

<?import rinde.stage.node.switch.RadioSwitch ?>
