Slider

Slider consists of a draggable thumb which you can slide over a track. Draggability enables the user to modify its value gradually, and every change is able to invoke an action. The action is, in most cases, a controller method which name is the same as the action's attribute value. Notice that this attribute is optional, so the action is invoked only if specified.

Import

<?import rinde.stage.node.slider.Slider ?>

XML

Obligatory attributes:
range
Optional attributes:
id
style_class
position_x
position_y
visible
enabled
action

CSS

Selector: slider

Example

<Slider range="255" action="value_changed" />
