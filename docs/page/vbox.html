VBox

VBox is the container for components stacked vertically.

XML

Optional attributes:
position_x
position_y
id
style_class
align (left/center/right)
spacing

CSS

Selector: vbox
Properties
position-x
position-y
width
height
visible
enabled
spacing
align (left/center/right)

Example

<VBox align="center" spacing="10">
    <Label text="1" />
    <Label text="2" />
    <Label text="3" />
</VBox> Defines vertically container with centered children and some space between each.
