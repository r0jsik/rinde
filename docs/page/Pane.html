Pane

Pane is a container for another components. It groups nodes, so their positions are relative to Pane's. The node doesn't align its children, so if you want to do it, use HBox, VBox or StackPane instead.

Import

Module has to be imported as <?import rinde.stage.node.pane.Pane ?>.

XML

Optional attributes:
id
style_class
position_x
position_y
width
height
margin
padding
visible
enabled

CSS

Selector: pane
Properties
position-x
position-y
width
height
margin
padding
visible
enabled

Example

<Pane position_x="20" position_y="30">
    <Label text="My position is: (20, 30)" />
</Pane>
