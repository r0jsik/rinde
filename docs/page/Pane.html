Pane

Pane is a container for another components. It groups nodes, so their positions are relative to their parent's. The Pane doesn't align its children, so if you want to do it, use HorizontalBox, VerticalBox or StackPane instead.

Import

<?import rinde.stage.node.pane.Pane ?>

XML

Optional attributes:
id
style_class
position_x
position_y
visible
enabled

CSS

Selector: pane
Properties
position-x
position-y
margin
padding
visible
enabled

Example

<Pane position_x="20" position_y="30">
    <Label text="My position is: (20, 30)" />
</Pane>
