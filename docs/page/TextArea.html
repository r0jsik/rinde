TextArea

TextArea allows the user to write something inside. It behaves like TextField, but is designed for long and multi-lined text. You can edit its content using the text property, and you can even specify a placeholder that will be shown when nothing typed in. To change its size, resize the background element.

<?import rinde.stage.node.field.TextArea ?>
