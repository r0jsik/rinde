StackPane

StackPane is a container that centers its children. Their position depends on pane size and position.

Import

Module has to be imported as <?import rinde.stage.node.pane.StackPane ?>.

XML

Optional attributes:
id
style_class
position_x
position_y
width
height
margin
padding
visible
enabled

CSS

Selector: stack-pane
Properties
position-x
position-y
width
height
margin
padding
visible
enabled

Example

<StackPane position_x="100">
    <Label text="Centered label" />
</StackPane>
