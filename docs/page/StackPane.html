StackPane

StackPane is a container that centers its children. If you want to align them differently, use HBox, VBox or Pane instead.

Import

Module has to be imported as <?import rinde.stage.node.pane.StackPane ?>.

XML

Optional attributes:
id
style_class
position_x
position_y
visible
enabled

CSS

Selector: stack-pane
Properties
position-x
position-y
margin
padding
visible
enabled

Example

<StackPane>
    <ImageView resource="background.png" />
    <Label text="Centered label" />
</StackPane>
