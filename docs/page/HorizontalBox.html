HorizontalBox

HorizontalBox is a container stacking components horizontally. You can align its children vertically (top, middle, bottom) and define space between them using the spacing property. It behaves similarly to VerticalBox.

Import

<?import rinde.stage.node.box.HorizontalBox ?>

XML

Optional attributes:
id
style_class
position_x
position_y
visible
enabled

CSS

Selector: horizontal-box
Properties
position-x
position-y
margin
padding
visible
enabled
align
spacing

Example

<HorizontalBox>
    <Label text="1" />
    <Label text="2" />
    <Label text="3" />
</HorizontalBox>
