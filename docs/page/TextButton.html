TextButton

TextButton allows the user to invoke action when clicked. The action is in most cases a controller method (function defined in the script), so the action attribute in a XML declaration has to be the same as the function's name. It looks like Label, but acts as Button.

Import

Module has to be imported as <?import rinde.stage.node.button.TextButton ?>.

Obligatory attributes:
action
