TextFlow

Allows the developer to show multi-lined text. Notice, that the only way to split the text is to insert new line characters \n wherever you want to wrap the line. You can edit its content by the text property. If you want to truncate the text to a specified width, use the textwrap module from the standard Python library.

Import

<?import rinde.stage.node.text.TextFlow ?>
