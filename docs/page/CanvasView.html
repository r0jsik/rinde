CanvasView

CanvasView displays a canvas on which you can draw. If width or height of the node changes, its content remains unmodified. You can access the canvas using get_canvas method. It is possible to change its content by invoking set_canvas method.

Import

<?import rinde.stage.node.view.CanvasView ?>

XML

Obligatory attributes:
width
height
Optional attributes:
id
style_class
position_x
position_y
visible
enabled

CSS

Selector: canvas-view
Properties
position-x
position-y
margin
padding
visible
enabled

Example

<CanvasView width="100" height="100" />
