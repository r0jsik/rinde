CanvasView

CanvasView is a display for a canvas on which you can draw. You can access the canvas from the script and draw on it by using predefined methods like draw_line or fill_rect. Notice, that if width or height of CanvasView changes, its content remains unmodified.

Import

Module has to be imported as <?import rinde.stage.node.view.CanvasView ?>.

XML

Obligatory attributes:
width
height
Optional attributes:
id
style_class
position_x
position_y
margin
padding

enabled
