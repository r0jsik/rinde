Button

Button allows you to invoke an action when clicked. The action is, in most cases, a controller method (function defined in the script), so the action attribute in a XML declaration has to be the same as the function's name. It differs from TextButton by having border and background.

Import

Module has to be imported as <?import rinde.stage.node.button.Button ?>.

XML

Obligatory attributes:
action
Optional attributes:
id
style_class
position_x

visible
enabled
text

Example
