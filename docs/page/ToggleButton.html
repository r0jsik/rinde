ToggleButton

The ToggleButton component looks like regular Button and behaves like CheckSwitch. The difference lies in possibility of changing its state and appearance when clicked (pressed and released). This component can also invoke its own method when clicked.

Import

<?import rinde.stage.node.button.ToggleButton ?>

XML

Obligatory attributes:
action
text
Optional attributes:
id
style_class
position_x
position_y
visible
enabled

CSS

Selector: toggle-button
Properties

position-y
margin
padding
visible
enabled
text

Structure

  • background
  • text

Example

<ToggleButton action="clicked" text="Toggle me!" />
