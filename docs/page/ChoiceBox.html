ChoiceBox

ChoiceBox is a widget which allows the user to select one of several options. It shows a disposer with an arrow informing that you can expand the ListView to select suitable option. Notice, that Option is the only node type the ChoiceBox can contain (it is possible to extend this class). It is possible to pass the placeholder as an argument, which will be shown when no option selected.

Import

<?import rinde.stage.node.list.ChoiceBox ?>

    <Option text="Hard" name="hard" />
