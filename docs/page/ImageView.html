ImageView

ImageView displays image from a file. Application has to find and be able to read indicated file. You can access and edit shown image from the script.

Import

Module has to be imported as <?import rinde.stage.node.view.ImageView ?>.

XML

Obligatory attributes:
resource
Optional attributes:
id
style_class
position_x
position_y
width
height
margin
padding
visible
enabled

CSS

Selector: image-view
Properties
position-x
position-y
width
height
margin
padding
visible
enabled

Example

<ImageView resource="image.png" />
