ImageView

ImageView displays an image loaded from file specified by the resource attribute (application has to find and be able to read indicated file). You can change its content using set_image method. If you modify dimensions of the ImageView, size of the image remains unchanged.

Import

<?import rinde.stage.node.view.ImageView ?>

XML

Obligatory attributes:
resource
Optional attributes:
id
style_class
position_x
position_y
visible
enabled

CSS

Selector: image-view
Properties
position-x
position-y
margin
padding
visible
enabled

Example

<ImageView resource="image.png" />
