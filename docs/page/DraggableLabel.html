DraggableLabel

DraggableLabel is a component used for displaying text on the stage. It differs from Label node only by ability to drag and move in the scene.

Import

Module has to be imported as <?import rinde.stage.node.text.DraggableLabel ?>.

Obligatory attributes:
text
