PasswordField

PasswordField allows the user to write inside the text, which will be replaced by dots. If you don't want to hide the text, use TextField instead. The plaintext is stored in the text property. You can set the placeholder, which will be shown when nothing typed in.

<?import rinde.stage.node.input.PasswordField ?>
