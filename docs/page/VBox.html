VBox

VBox is a container stacking components vertically. You can align its children horizontally (left, center, right) and define space between them using the spacing property. It behaves similarly to HBox.

Import

<?import rinde.stage.node.box.VBox ?>

XML

Optional attributes:
id
style_class
position_x
position_y
visible
enabled

CSS

Selector: vbox
Properties
position-x
position-y
margin
padding
visible
enabled
align
spacing

Example

<VBox>
    <Label text="Example label" />
    <Label text="Label" />
    <Label text="Another label" />
</VBox>
