VBox

VBox is a container which stacks components vertically, while you can align its children horizontally. The spacing property defines margin between children. It behaves similarly to HBox, but uses different orientation.

Import

Module has to be imported as <?import rinde.stage.node.box.vbox.VBox ?>.

XML

Optional attributes:
id
style_class
position_x
position_y
width
height
margin
padding
visible
enabled
spacing
align

Example

<VBox align="center" spacing="10">
    <Label text="1" />
    <Label text="2" />
    <Label text="3" />
</VBox>
