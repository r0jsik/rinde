RadioBox

RadioBox allows the user to switch between many options, but both of them can't be selected together. Options are grouped by the common group attribute. Every node in the group has to be named differently. You can get selected option's name using the groups field in a controller.

Import

Module has to be imported as <?import rinde.stage.node.input.RadioBox ?>.

Obligatory attributes:
group
name
