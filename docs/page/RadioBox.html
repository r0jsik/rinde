RadioBox

RadioBox allows you to switch between many options, but both of them can't be selected together. Options are grouped by common group attribute. Every node in the group has to be named differently. You can access selected option in Controller.

Import

Module has to be imported as <?import rinde.stage.node.input.RadioBox ?>.

XML

Obligatory attributes:
group
name
Optional attributes:
id
style_class
position_x
position_y
width
height
margin
padding
visible
enabled
text
selected

Example

<RadioBox text="Easy" group="difficulty" name="easy" selected="true" />
