Pages

The Pages is a special container that stores Page components. The main advantage of this node is that you can easily switch between visibility of its children; when you set visibility of one child to true, another Pages in the same group hide.

Import

<?import rinde.stage.node.page.Pages ?>

XML

Obligatory attributes:
group
Optional attributes:
id
style_class
position_x
position_y
visible
enabled
select

CSS

Selector: pages
Properties
position-x
position-y
margin
padding
visible
enabled

Example

<Pages group="page" select="page_2">
    <Page name="page_1">
        <Label text="This Page will not be shown by default" />
    </Page>
    
    <Page name="page_2">
        <Label text="I am the selected Page" />
    </Page>
</Pages>
