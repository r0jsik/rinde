# rinde
Rinde is amazing Python library which can be used for creating GUI based on XML and CSS.

*Work in progress.*

Requires:
<ul>
    <li>Python 2.7 (for now)</li>
    <li>cssutils</li>
	<li>pygame (for now)</li>
</ul>
